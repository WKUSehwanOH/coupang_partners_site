# Coupang Partners Landing Site

GitHub Pages 배포용 사이트입니다.

## 업로드 방법
1. 이 ZIP 파일의 압축을 풉니다.
2. GitHub 저장소를 새로 만듭니다.
3. 압축을 풀었을 때 보이는 `index.html`, `style.css`, `script.js`, `.nojekyll`, `README.md` 파일을 저장소의 가장 바깥(root)에 업로드합니다.
4. GitHub 저장소에서 Settings > Pages로 이동합니다.
5. Source를 `Deploy from a branch`로 선택합니다.
6. Branch는 `main`, Folder는 `/root`로 선택 후 Save합니다.
7. 1~5분 후 `https://깃허브아이디.github.io/저장소이름/` 주소로 접속합니다.

## 주의
- `index.html`이 폴더 안에 들어가 있으면 GitHub Pages가 제대로 열리지 않을 수 있습니다.
- 쿠팡 파트너스 링크는 `index.html` 안의 버튼 링크를 본인 링크로 교체하세요.
